## 🧾 What We Are Not Paying For (Yet)

- **DDoS Protection Standard** (~$2,944/mo): Deferred to Phase 2. Front Door Premium WAF covers Layer 7.
- **Customer-Managed Keys (CMK)**: Platform-managed encryption used. CMK adds Key Vault Premium HSM costs.
- **Multi-region DR**: Single-region per RFQ exclusion. DR would add ~$3,000-4,000/mo for geo-redundant secondaries.
- **Azure Chaos Studio**: Resilience testing deferred to Release 1.1.
- **Premium SSD v2 / Ultra Disks**: Standard Premium SSD P30 sufficient for current workloads.
- **Entra External ID Go-Local add-on**: EU data locality guarantee ($0.02/MAU = $300/mo) optional if GDPR review requires it.
- **Log Analytics Commitment Tier**: Pay-as-you-go at ~15 GB/mo; commitment tiers become cost-effective at >100 GB/day.

### Assumptions & Uncertainty

- **Exchange rate**: $1 USD = €0.92 EUR (March 2026 estimate; fluctuation affects EUR totals)
- **Hours/month**: 730 hours (365.25 days ÷ 12 × 24)
- **Network egress**: Minimal (<50 GB/mo) for MVP; scaling with user growth
- **Storage growth**: 200 GB Blob + 256 GB Files + 256 GB Disk at MVP; projected 500 GB→1.5 TB by 2027
- **Log ingestion**: ~15 GB/mo production initially; may reach 50-100 GB/mo at scale
- **Redis utilization**: 128 GB requirement from RFQ taken at face value; actual working set may be smaller
