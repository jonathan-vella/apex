### Month-over-Month Projection

### Key Design Decisions Affecting Cost

| Decision                     | Cost Impact                                        | Business Rationale                                            | Status   |
| ---------------------------- | -------------------------------------------------- | ------------------------------------------------------------- | -------- |
| B1 instead of S1 App Service | Keeps spend roughly €50/month lower                | Current concurrency target does not justify Standard features | Required |
| Basic instead of S0 SQL      | Keeps spend roughly €8/month lower                 | Order volume target is low                                    | Required |
| LRS instead of GRS storage   | Reduces storage cost                               | Single-region DR model accepted for MVP                       | Required |
| No private endpoints         | Avoids additional recurring charges and complexity | Security model is identity-first for MVP                      | Optional |

## 🧾 What We Are Not Paying For (Yet)

- Private endpoints for SQL, Storage, or Key Vault.
- A second region with duplicate compute and data services.
- CDN or Redis cache layers.
- Standard or Premium App Service features such as autoscale and deployment slots.
- Additional non-production environments.
