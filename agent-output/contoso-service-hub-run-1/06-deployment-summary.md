### Prerequisites Before Deployment

1. **EU Data Boundary tenant configuration** — Must be set at tenant creation (cannot retrofit)
2. **Subscription-level Azure Policy assignments** — Apply governance policies from `04-governance-constraints.json`
3. **Service principal or managed identity** for deployment with Contributor + RBAC Admin roles
4. **SSH key pair** for VM authentication
5. **PostgreSQL admin credentials** stored in Key Vault

## 📝 Post-Deployment Tasks

1. Configure AKS kubectl access and deploy application workloads
2. Set up Application Gateway backend pool with AKS ingress controller IP
3. Configure APIM APIs and policies
4. Create DNS records in the public DNS zone
5. Configure Entra External ID tenant with FIDO2/TOTP MFA only
6. Set up monitoring alerts and dashboards in Azure Monitor
7. Validate private endpoint connectivity for all data services
8. Run GDPR compliance audit on deployed resources
