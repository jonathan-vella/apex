## 🧾 What We Are Not Paying For (Yet)

- **Multi-region active-active DR** — excluded from RFQ scope (§4.1). Would add ~$5,000–$8,000/month
- **DDoS Protection Standard** — relying on Front Door WAF basic DDoS. Standard plan would add ~$2,944/month
- **Customer Managed Keys (CMK)** — using platform-managed keys. CMK with dedicated HSM would add ~$500/month
- **APIM Premium (zone-redundant)** — using Standard v2 (no AZ). Premium would add ~$2,800/month
- **Microsoft Defender for Containers** — not yet enabled. Would add ~$7/agent/month per AKS node
- **Azure Firewall** — not required for current architecture. Would add ~$900/month if needed for egress filtering

### Assumptions & Uncertainty

- **Log ingestion**: Estimated at ~5 GB/day based on 15 services across 3 environments; actual volume depends on application verbosity
- **Network egress**: Minimal EU-internal egress assumed; Front Door serves most external traffic
- **Storage growth**: 200 GB initial, growing ~50 GB/quarter in production for content and transaction records
- **Redis tier**: E100 pricing assumes swedencentral availability; verify region-specific pricing at deployment
- **Exchange rate**: €1 = $1.08 USD for EUR/USD conversions (planning assumption, not hedged)
