## 🧾 What We Are Not Paying For (Yet)

- Full multi-region disaster recovery
- Private endpoint data transfer and DNS operational overheads in detail
- Explicit API call volume, Front Door request volume, and Service Bus operation metering
- Dedicated CIAM licensing and integration cost

### Assumptions & Uncertainty

- The bulk estimate uses direct service and SKU matches where possible, but some platform services resolved to base meters rather than full usage-driven meters.
- The AKS worker baseline is represented by a VM proxy that resolved to a Low Priority meter, which underestimates production on-demand cost.
- The Redis line item is a pricing upper-bound proxy for the 128 GB enterprise-class cache target.
