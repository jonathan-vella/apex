## 🧾 What We Are Not Paying For (Yet)

- Azure Bastion for secure VM access (~$139/mo Standard SKU)
- Log Analytics workspace + VM agent (~$5-15/mo depending on data volume)
- Azure Backup for VM (~$10/mo for single VM)
- Application Gateway / WAF for HTTPS termination (~$180/mo)
- DDoS Protection Standard (~$2,944/mo — not recommended for dev)
- Additional VMs for high availability (~$37/mo per VM)

### Assumptions & Uncertainty

- VM runs 24/7 (730 hours/month) — deallocation reduces compute cost proportionally
- Negligible data transfer — LB data processing at $0.005/GB is not modeled (<1 GB expected)
- Windows Server license included in VM price (PAYG) — no hybrid benefit applied
- No premium storage IOPS charges — Standard SSD E10 includes 500 IOPS baseline
