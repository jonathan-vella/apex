Use the controls inside the explorer to filter by category, search for nodes, and click any node to see its connections.

## Node Categories

| Category    | Color  | Description                          |
| ----------- | ------ | ------------------------------------ |
| Agent       | Blue   | Top-level orchestrator agents        |
| Subagent    | Indigo | Delegated helper agents              |
| Skill       | Green  | Reusable domain knowledge modules    |
| Instruction | Amber  | Enforcement rules (.instructions.md) |
| Prompt      | Pink   | Agent prompt templates               |
| Validator   | Purple | CI check validators                  |
| CI Workflow | Red    | GitHub Actions workflows             |
| Gate        | Orange | Workflow approval gates              |
| MCP Server  | Cyan   | External tool servers                |
